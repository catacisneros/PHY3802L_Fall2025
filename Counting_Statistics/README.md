# Counting-Statistics

Analysis of Geiger-Mueller tube counting statistics data for Intermediate Physics Lab (FIU, Fall 2025).

## Overview

Statistical analysis of radiation counting experiments including:
- Plateau study (optimal operating voltage)
- Low/high statistics (Poisson vs Gaussian distributions)
- Background correction
- Distance dependence (inverse square law)

## Usage

```bash
python counting_statistics_analysis.py
```

## Requirements

Python 3, numpy, LT.box
